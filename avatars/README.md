# Feed avatars

Drop a square image here named after the feed's rkey, and `yarn publishAll`
uploads it automatically:

| File | Feed |
|---|---|
| `md-parivaar.png` (or `.jpg`) | MD Parivaar |
| `k4m2a.png` | K4M2A |
| `coseeker.png` | CoSeeker |

**Format:** PNG or JPEG, square (e.g. 1000×1000). Keep it under ~1 MB.

If no file exists for a feed, any avatar already on the published record is left
untouched.
